# a django app
技术栈：
- 后端：python3+django+redis+mysql+nginx
- 前端：html5+css3+js+bootstrap+ajax
功能：
- 1.用户注册，登录，用户资料页
- 2.书籍热度排行，收藏，阅读功能
- 3.商家后台
- 4.支持响应式
- todo:order模块
目前为了方便使用redis作为缓存及消息中间件，后续消息中间件会改为rabbitMQ实现；
